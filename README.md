## ğŸ“¦ Deployment

### Deploy to Vercel

1. Push code to GitHub (already done!)
2. Go to [vercel.com](https://vercel.com)
3. Import this repository
4. Click "Deploy"

### Configure Base MINI App

1. Visit [base.dev/build](https://base.dev/build)
2. Sign in with your Farcaster/Base account
3. Generate `accountAssociation` credentials
4. Update `minikit.config.ts` with your credentials
5. Push changes to GitHub

### Publish

1. Preview at [base.dev/preview](https://base.dev/preview)
2. Post your app URL in Base app to publish
3. Share with friends!
